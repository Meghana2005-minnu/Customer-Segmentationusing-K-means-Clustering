## Objective
Segment customers based on purchasing behavior using K-Means clustering.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data preprocessing
- Elbow method
- K-Means clustering
- Customer visualization
- Business insights
